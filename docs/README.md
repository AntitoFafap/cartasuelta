# Carta Suelta — Next.js (plantilla)

Proyecto Next.js mínimo creado en la carpeta `pokemon`.

Requisitos:
- Node.js v18+ (tú tienes v24 — OK)

Comandos útiles:

- Instalar dependencias: `npm install`
- Ejecutar en desarrollo: `npm run dev`
- Construir para producción: `npm run build`
- Iniciar servidor de producción: `npm start`

Si quieres que inicie el servidor de desarrollo ahora, dime y lo lanzo. ¡Listo para continuar!
